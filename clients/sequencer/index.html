Pour participer à la performance musicale collective:
1) Connectez-vous au réseau Wi-Fi: loops (pas de mot de passe)
2) Sur votre navigateur Web tapez l'adresse: http://loops.play (ou http://192.168.2.24)
Conseils:
  • Si vous utilisez un smartphone:
    • tenez le en mode "paysage"
    • configurez le de façon à ce que votre écran ne s'éteigne que toutes les 3 à 5 minutes
    • pour le domaine loops.play désactivez la traduction automatique des pages Web de l'anglais vers le français

  • Si vous rencontrez des problèmes:
    • rafraîchissez votre page Web
    • installez et utilisez le navigateur Google Chrome (...Loops ne semble pas fonctionner sous Apple iOS - Safari)
    • vérifiez que votre appareil est compatible avec le Wi-Fi N (norme IEEE 802.11n) utilisé par le réseau local loops
